# Portfolio images

Put your images here. They are used like this:

- **Resume (hero photo):** Edit `src/pages/ProfilePage.jsx` and change `image/IMG_0456.JPG` to `image/your-photo.jpg`.
- **Projects banner:** Edit `src/pages/ProjectsPage.jsx` and change `image/IMG_0175.JPG` to `image/your-banner.jpg`.

Use the **exact filename** (e.g. `IMG_0456.JPG`). Works with `.jpg`, `.jpeg`, `.png`, `.webp`.
